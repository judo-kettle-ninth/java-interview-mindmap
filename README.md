# Java 面试知识体系思维导图

> ProcessOn × 小林coding × JavaGuide × 卡码三站 知识融合

基于 ProcessOn 上的 Java 面试知识体系思维导图（5384 节点），融合了三站（小林coding、JavaGuide、卡码）共 1,189 篇文章 / 16,168 个标题的精华内容，最终形成 **10,251 节点** 的完整知识树。

- **多线程与并发**：JMM、锁、线程池、AQS、CAS、ThreadLocal
- **JVM**：内存模型、GC、类加载、调优
- **MySQL**：索引、事务、锁、优化、分库分表
- **Redis**：数据结构、持久化、集群、缓存设计
- **网络**：TCP/IP、HTTP、HTTPS、DNS
- **操作系统**：进程线程、内存管理、文件系统
- **Spring 全家桶**：IoC、AOP、MVC、Boot、Cloud、Security
- **分布式**：CAP、一致性、分布式事务、微服务
- **Kafka**：架构、存储、可靠性、性能
- **Docker / Kubernetes**
- **设计模式**
- **C++ 面试**
- **大模型 / AI / LLM**：Transformer、RAG、Agent、微调

## 使用

打开 `index.html` 即可在浏览器中查看交互式思维导图：

- **展开/收起**：点击节点上的圆圈
- **缩放**：鼠标滚轮 / 触控板捏合
- **平移**：拖拽空白区域
- **适应屏幕**：点击右下角工具栏的适应按钮
- **深色模式**：跟随系统主题
